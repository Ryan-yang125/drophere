# DropHere roadmap

DropHere is a free, community-oriented deployment tool for AI coding agents.
The roadmap follows recurring deployment failures and contributor demand.

## Current focus

- Keep the Agent Skill small, deterministic, and easy to inspect.
- Detect common static build outputs automatically.
- Stop likely secrets before upload and make the public-upload boundary clear.
- Verify the deployed URL, entry page, assets, and indexing behavior.
- Maintain the hosted free service within published fair-use limits.

## Candidate work

- More framework-specific deployment checks.
- Machine-readable CLI output for agent orchestration.
- Reproducible end-to-end Skill benchmarks.
- A documented self-hosting path for the service components.
- Community adapters for additional agent runtimes.

Issues with a minimal reproduction are the strongest input for roadmap
priority.
